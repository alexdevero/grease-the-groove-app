### Install

Clone this repo
```
$ git clone https://github.com/alexdevero/grease-the-groove-app.git
```

### Usage
#### Run the app

```
npm run start
```
or
```
yarn run start
```

#### Build the app (automatic)

```
npm run package
```
or
```
yarn run package
```

#### Test the app (after `npm run build` || `yarn run build`)
```
npm run prod
```
```
yarn run prod
```
